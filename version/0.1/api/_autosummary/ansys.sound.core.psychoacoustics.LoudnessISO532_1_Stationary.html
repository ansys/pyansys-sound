LoudnessISO532_1_Stationary#

class ansys.sound.core.psychoacoustics.LoudnessISO532_1_Stationary(signal=None)#

Computes ISO 532-1 loudness for stationary sounds.

This class computes the loudness of a signal following the ISO 532-1 standard for stationary sounds.
